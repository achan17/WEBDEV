Cautionary Code

A teal rectangle.
Notice the difference in background color? While cool, it was implemented in the HTML section which violates the Separation of Concerns principle. Even this section which to the untrained eye seems fine has some practices which can be problematic. Notice the bold and italic words, while both styles were used elsewhere in this page, this page I used physical tags rather than logical which was used in the other pages. Along with those, there are the cautionary advices such as the div and span HTML elements I used here which are not inherently bad until you use too many of these elements to the point where the page becomes littered with divs and spans. However, the biggest issue in this section is at problematic. In here not only did I use a span, but I also combined it with styling to change the font size which violates Separation of concerns. In short, if you inspect this div element in your browser, there are numerous practices that may be used by others but are advised against.

If you would like to know more about web developing and the practices mentioned such as Separation of Concerns, UCSD's CSE 134B class: Web Client Languages by Professor Powell is a great place to start.

E = m ⁢ c 2

Now that formula may just seem like a simple font adjustment, but Einstein's famous formula was actually done using MathML which allow html to write out math formulas and equations, even some of the most complicated formulas could be written out with enough patience. Unbe- knownst to many, MathML has been a part of the HTML specification for a while but developers could not make use of it until recently and that is because for about 80 different versions, google chrome did not support the features of MathML meaning for however long the versions lasted, Chrome would not make its use available for the browser until recently. A little birdie told me that it was not Chrome's development team that implemented the MathML support but someone outside of the company who asked to push his implementation onto the main branch and google accepted the push.
